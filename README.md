Project title: Linked-List Elevator
Project Overview: This project was implimented to simulate a real-world elevator system by using a doubly linked list where each node represents a floor of a building that a person can go to.
The doubly-linked list was used to simulate how each elecator moves and communicates with the the floors around it. Every floor has a refrence to the floor above and floor below using pointers,
this allows the elevator to travel up and down in constant time between adjacent floors. The application handles different
elevators behaviors including: accepting request, moving up and down between floors, accessing current floor, accessing queue of pending request and more. The goal of this projecg was to understand and demonstrate node-based data structures, practice memory allocation using pointer-style insertion and traversal logic, and applying an abstract data structure to a real world simulation.
Technologies used: Language: C++, IDE/Editor: VS Code, Version Control: Git and Github
How to Run: This project can be run by clonning the repository and compiling it from the command line: git clone https://github.com/Tnwokike/Linked-list-elevator.git
cd Linked-list-elevator. It can also be run through opening the project floor in your ide and run the main class directly with the run command.
